DESIGN

Design mockup for MicroMain Global.

Project Description & Results.

THE CLIENT

MicroMain is the current company that I work for, this is an early mock up that I made for the global project we are releasing this year.

THE PROPOSAL

We wanted a very simplistic feel with touches of fun colors without getting too chaotic.

THE RESULTS

I made each application have it's own color while the main colors stayed white, grey, and blue. For "added flair" I created a block spill for the welcome page that contained the colors with a faded undertone.
